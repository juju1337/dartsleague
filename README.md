This is a very lightweight web based darts tournament management system. 
It uses PHP and plain csv files to manage players and matches.
Results are entered after the matches and statistics are calculated.

Note that this is work in progress and specifically designed for our own tournament format.

Instructions:
- Specify admin password in index.php ($admin_password = '1234'; // Change this to your desired password)
- Copy all files to webserver with PHP support enabled.
- Navigate to index.php
- Login and begin to enter players and set up a tournament
